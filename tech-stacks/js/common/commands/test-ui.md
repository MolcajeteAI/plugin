# Open Vitest UI

Launch the Vitest graphical user interface for interactive testing.

Execute the following command:

```bash
npm run test:ui
```

This runs:
```bash
vitest --ui
```

**UI Features:**
- Visual test explorer
- Real-time test results
- Filter and search tests
- View test code and output
- Re-run individual tests
- Module dependency graph

**Browser Access:**
Opens automatically at `http://localhost:51204/__vitest__/`

Reference the **vitest-configuration** skill.
